VM安装虚拟机教程及镜像制作

1.VM虚拟机安装

1.1 windows

1.根据系统版本下载对应的VM客户端即可

下载地址：https://www.vmware.com/products/workstation-pro.html

2.安装步骤：

2.1.双击.exe文件， 开始安装。

2.2.选择安装位置，安装在一个大一点的盘，安装路径尽量不要有中文

2.3.安装成功后，会要求输入密钥,下面给出一些备选，基本都是永久激活的

2.4 VM客户端安装完毕

3.使用步骤

3.1 打开VM

3.2 点击 文件->新建虚拟机

3.3 选择“典型”，点击下一步，选择“安装程序光盘映像文件”，路径为已经准备好的ISO镜像文件所在的位置

3.4 起个名字，然后会出现虚拟机安装位置选择（此位置对应于所要安装的ISO镜像系统的存储位置，以后都会去这个路径下加载系统,最好新建个文件夹并记住，免得忘了去哪找）

3.5 接下来会选择磁盘大小等系统配置，这个按所给出的建议值来设置就好，可以比值大1.5倍，最后安装完毕，系统启动

1.2 Mac

3.镜像的制作

VM虚拟机原理是硬件虚拟化，即为了使所安装的虚拟系统可以正常运行，对计算机底层硬件资源进行虚拟化。其好处在于虚拟系统的运行独立。虚拟机每次重新启动都是对上述安装过程中所安装的系统进行2重复启动，每次启动的状态为系统初始状态，也就是说你的操作将会在重启后丢失，如果想保留这些操作，有两种方法（1）用完不关机，选择挂起，可以理解为将系统暂停，状态序列化保存在文件里。下次点击继续使用即可 （2）在对系统操作后，想要保存现状态，则将虚拟机重新导出为一个新的ISO，步骤为：点击“虚拟机”->“管理”->“克隆“。

4.平时使用

无特别提醒。
