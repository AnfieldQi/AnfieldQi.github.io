2. 类属性和实例属性

  1. 给实例绑定属性的方法是通过实例变量，或者通过self变量

  3. 优先级：实例属性>类属性
  4. 对于类属性，每个实例都可以访问和改变

3.继承和多态

？？算了，重复一下是啥意思吧

继承
  1. 子类继承父类
多态

  1. 建立在继承的基础上，同一接口提供多种不同的实现形式

  2. 例子：

    //示例来源于廖雪峰的网站
class Animal(object):
    def run(self):
        print('Animal is running...')
   
//啥也不干，猫狗就有了run，不过如果猫狗有自己的run，就会覆盖掉父类的run
//当然了这个猫和狗使用isinstance判断类型也属于Animal
class Dog(Animal):
    pass
class Cat(Animal):
    pass
       
dog = Dog()
dog.run()
   
cat = Cat()
cat.run()
   
//啥是多态啊？建立run_t，接受Animal类型的变量
def run_twice(animal):
    animal.run()
    animal.run()
//我们再写一个大鹅，继承于Animal
class Dae(Animal):
    def run(self):
        print('gagaga')
//不用修改rt，直接可以给大鹅用上。
run_twice(Dae())

    由于Animal类型有run()方法，因此，传入的任意类型，只要是Animal类或者子类，就会自动调用实际类型的run()方法，这就是多态。 对于一个变量，我们只需要知道它是Animal类型，无需确切地知道它的子类型，就可以调用run()方法，这就是多态真正的威力：调用方只管调用，不管细节，而当我们新增一种Animal的子类时，只要确保run()方法编写正确，不用管原来的代码是如何调用的。

  1. 开闭原则：

  对扩展开放：允许新增子类

  对修改封闭：不需要修改依赖Animal类型的run_twice()等函数

5.__ slots __的使用

  1. 涉及到为实例绑定属性和方法

    >>> def set_age(self, age): # 定义一个函数作为实例方法
...     self.age = age
...
>>> from types import MethodType
>>> s.set_age = MethodType(set_age, s) # 给实例绑定一个方法
>>> s.set_age(25) # 调用实例方法
>>> s.age # 测试结果
25

    这种的话其他实例是不可能绑定的，如果绑定可以给class，Student.set_score = set_score

  2. 动态绑定的时候，如果要限制绑定的属性，就要__ slots __

    //作用范围：只对当前类实例有用，对父子类没用
class Student(object):
    __slots__ = ('name', 'age') # 用tuple定义允许绑定的属性名称

6.@property的使用

  1. 用法：将一个方法变为一个属性来用

    class Student(object):
   
    @property
    def score(self):
        return self._score
   
    @score.setter
    def score(self, value):
        if not isinstance(value, int):
            raise ValueError('score must be an integer!')
        if value < 0 or value > 100:
            raise ValueError('score must between 0 ~ 100!')
        self._score = value
//把一个getter方法变成属性，只需要加上@property就可以了，@property本身又创建了另一个装饰器@score.setter，负责把一个setter方法变成属性赋值
//还可以定义只读属性，只定义getter方法，不定义setter方法就是一个只读属性

7.枚举类

  1. 与常量的定义有关,一般用大写

  2. 最好用枚举

    //普通
from enum import Enum
Month = Enum('Month', ('Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'))
   
//自定义
from enum import Enum, unique
@unique //检查有没有重复值
class Weekday(Enum):
    Sun = 0 # Sun的value被设定为0
    Mon = 1
    Tue = 2
    Wed = 3
    Thu = 4
    Fri = 5
    Sat = 6
//引用方法
//既可以用成员名称引用枚举常量，又可以直接根据value的值获得枚举常量
Weekday.Mon.value
Weekday['Tue']
Weekday(1)

    Enum可以把一组相关常量定义在一个class中，且class不可变，而且成员可以直接比较。
