固件信息熵计算

Written by anfieldqi with ♥ on May 17, 2020 in IoT安全 , 1050 words

信息熵

  1. 熵（英语：entropy）是接收的每条消息中包含的信息的平均量

  2. 规则：熵值越高，所含信息越多；熵值越低，所含的信息也越低，或者说是所含信息复杂或简单，信息熵是信息量的期望（均值），它不是针对每条信息，而是针对整个不确定性结果集而言，信息熵越大，事件不确定性就越大。单条信息只能从某种程度上影响结果集概率的分布

1. 不确定性事物的衡量

  1. 不确定性事物必须找到一种抽象化，公式化的手段来表示

  2. 例子：

    P=[p1,p2,p3,p4]
p1:阴 p2:晴 p3:雨 p4:雪
   
对于以下三种描述：
1.今天周日
2.明天下雨
3.后天放假
   
显然2的信息量更大，因为其可以使不确定性发生在P3的几率更大，而其他两条基本信息量为0

2.信息冗余与信息量存储

在上面的例子中，已经有了P表示四种天气情况，下面来思考如何进行存储

  1. 常见方法：00，01，10，11分别表示四种状态。再考虑到数据的压缩，如果身处北极，一定把00设置为P3

  2. 计算公式：（n天数据）

    Pi：第i件事发生的概率 F（Pi）：存储空间因子

  3. **F（Pi）**的计算形式

    规则：概率大的事件对应小的存储空间，最常用的是反比例函数，即1/Pi,但是考虑到正态分布，使用取对数：

    a作为底数，可以取2（处理2bit数据），10，e（处理正态分布相关的数据）

    实质就是根据真实的分布的一个表示系统不确定性的最小代价，某种意义上是信息存储所需空间

1. 计算规则

1.Shannon熵分析方法（与上边介绍的公式相同）

import math
import codecs

def shannon(self, data):
    '''
    对给定的数据块执行香农熵分析。
    '''
    entropy = 0
 
    if data:
        length = len(data)
        seen = dict(((chr(x), 0) for x in range(0, 256)))  # ascii码（含后128个扩展码）
        for byte in data:
            seen[byte] += 1
 
        for x in range(0, 256):
            p_x = float(seen[chr(x)]) / length  # 每个ascii字符占总长度的比率
            if p_x > 0:
                entropy -= p_x * math.log(p_x, 2)
    return (entropy / 8) #这个是为了将结果固定在01之间

if __name__ == '__main__':
    with codecs.open('1.txt', 'r', encoding='utf-8', errors='ignore') as keyword_search:
        data=keyword_search.read()
        shannon(data)

2. gzip信息熵分析方法

import zlib

def str2bytes(string):
    '''
    用于Python2和Python3字符串之间的交叉兼容性。
    '''
    if isinstance(string, type('')) and PY_MAJOR_VERSION > 2:
        return bytes(string, 'latin1')
    else:
        return string
 
def gzip(self, data, truncate=True):
    '''
    基于zlib压缩比计算信息熵。
    这比shannon分析更快，但是不是很准确。
    '''
    # 求信息熵的方法：zlib压缩大小/原始大小
    e = float(float(len(zlib.compress(str2bytes(data), 9))) / float(len(data)))
 
    if truncate and e > 1.0:
        e = 1.0
        
if __name__ == '__main__':
    with codecs.open('1.txt', 'r', encoding='utf-8', errors='ignore') as keyword_search:
        data=keyword_search.read()
        gzip(data)
