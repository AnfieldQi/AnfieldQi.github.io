Linux设置定时任务

Written by anfieldqi with ♥ on July 23, 2020 in 教程工具 , 274 words

Crontab

1.简介

crontab是cron table的简写，用来在LINUX上设置定时任务是cron的配置文件，也可以叫它作业列表.cron启动后其自动启动。

2.安装

  1. 安装cron

    apt-get install cron

  2. 启动cron

    systemctl start cron

  3. 查看cron状态

     systemctl status cron

3.相关命令

  1. 启动服务

    systemctl start crond

  2. 关闭服务

    systemctl stop crond

  3. 重启服务

    systemctl restart crond

  4. 重新载入配置

     systemctl reload crond

  5. 查看状态

    systemctl status crond

4.定时任务的设置格式

  1. 打开任务设置面板

    crontab -e

  2. 删除任务时程表

    crontab -r

  3. 列出任务时程表

    crontab -l

  4. 任务设置格式

    * * * * * cmd
*    *    *    *    *
-    -    -    -    -
|    |    |    |    |
|    |    |    |    +----- 星期中星期几 (0 - 7) (星期天 为0)
|    |    |    +---------- 月份 (1 - 12) 
|    |    +--------------- 一个月中的第几天 (1 - 31)
|    +-------------------- 小时 (0 - 23)
+------------------------- 分钟 (0 - 59)
