文件的读写

f = open('path', 'r', encoding='', errors='ignore')

f = open('/Users/michael/test.txt', 'r')
f = read() //读入内存存为str对象
f.close() //关掉

//最好是调用finally，不然万一read出了错就关不掉了
//这里恰好复习一下异常处理，如果open的路径无效，那么就raise了异常，就先去找有没有对应异常处理，没有的话执行finally（让错误程序释放一些资源的功能写在这）,finally又异常了，完了，程序停了
try:
    f = open('/path/to/file', 'r')
    print(f.read())
finally:
    if f:
        f.close()
//最好还是一个try catch一个错误

//一般用不着，直接用with open就行了

OS操作文件目录

把两个路径合成一个时，不要直接拼字符串，而要通过os.path.join()函数，这样可以正确处理不同操作系统的路径分隔符 ， 要拆分路径时，也不要直接去拆字符串，而要通过os.path.split()函数，这样可以把一个路径拆分为两部分，后一部分总是最后级别的目录或文件名

func context
os.path.abspath('.') 绝对路径
os.path.join（path,file） 获取file完整路径
os.path.split（path） 拆分路径
os.path.splitext() 直接获取最后的扩展名
os.mkdir() 创建新目录
os.rmdir() 删除目录
os.remove（path） 删除文件
os.rename（old,new） 重命名
os.path.isdir(x)/isfile(x) 判断类型

序列化

  1. 定义：将变量从内存中变成可存储或传输的过程，序列化后内容可以写入磁盘

模块：

  1. pickle

    1. pickle.dumps():对象序列化为bytes,
    2. pickle.dump(d,fileobj) ：直接写入文件对象file
    3. pickle.loads()：字符串反序列化
    4. pickle.load(fileobj)：将内容在file obj中读出

  2. json:不同语言之间传递，比如java和python

    1. json.dumps():返回一个json格式的串
    2. json.dump(d，file):直接写入文件对象file
    3. json.loads():字符串反序列化
    4. json.load(fileobj):直接在file obj反序列化出对象
