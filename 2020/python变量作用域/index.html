python 变量及其作用域

四个作用域

LEGB原则：

  1. L(local)：局部作用域
  2. E(Enclosing)：闭包函数外的函数中
  3. G(global)：全局作用域
  4. B(built-in)：内建作用域
简单规则

  1. 查找规则：L–>E–>G–>B

  2. 作用规则：

    1. def/class/lambda 可改变作用域
    2. 不可改变：if/elif/else/ try/except for/while等
    3. attention：在def/class/lambda中进行内部赋值操作的话，变成局部作用域

  3. 在 def/class/lambda内进行赋值，就变成了其局部的作用域，局部作用域会覆盖全局作用域，但不会影响全局作用域。

g = 1 #全局的
def fun(): 
     g = 2 #局部的 
     return g 
     
 print fun() 
 # 结果为2
 print g 
 # 结果为1
  1. 在局部作用域引用全局变量需要进行标识global,修改后全局变量也会改变
#file1.py
var = 1
def fun():
    print var #报错，因为var=200赋值语句，var被判为局部
    var = 200
print fun()

#file2.py
var = 1
def fun():
    var = var + 1 #这个也一样
    return var
print fun()

#这两个需要修改的时候必须加global，但是不需要修改的时候（例：print）的时候不需要global声明
  1. 关于修改与否更简单的例子
g = 1 #全局的
def fun(): 
     g = 2 #局部的 
     return g 
     
 print fun() 
 #这样修改不需要global，直接覆盖，在函数作用域有效，不影响全局变量

g = 1 #全局的
def fun(): 
     print(g)
     g = 2 #局部的 
     return g 
     
 print fun() 
 #这样就需要修改，在print前面加global g,因为判断为局部变量了，但是局部变量没赋值，print不知道去print谁（这也就是规则里说的引用全局变量），加上global后修改后全局也该

tips: 外部函数的变量可以被内部函数所使用，但不能被内部函数修改，若要修改需要添加关键字nonlocal，跟上面global一样

闭包closure

闭包的定义：如果在一个内部函数里，对在外部函数内**（但不是在全局作用域）**的变量进行引用，那么内部函数就被认为是闭包(closure)

1. 函数嵌套/闭包中的作用域
a = 1
def external():
    global a
    a = 200
    print a

    b = 100
    def internal():
        # nonlocal b
        print b
        b = 200
        return b

    internal()
    print b

print external()
